# E-Commerce Sales & Customer Insights Dashboard

A data analytics dashboard built using **Power BI** to analyze e-commerce sales performance and customer behavior.  
The project connects to a **PostgreSQL database** and uses **SQL, DAX, and Power Query** to transform and analyze order data.

---

## 📊 Project Overview

This dashboard provides insights into:

- Sales performance
- Customer acquisition trends
- Returning vs new customers
- Repeat customer rate
- Geographic distribution of customers
- Order activity and revenue patterns

The goal of the project is to demonstrate **data modeling, SQL querying, and business analytics using Power BI**.

---

## 🛠 Tech Stack

- **Power BI** – Dashboard creation and data visualization  
- **PostgreSQL** – Database connection and data storage  
- **SQL** – Data extraction and preparation  
- **Power Query** – Data cleaning and transformation  
- **DAX (Data Analysis Expressions)** – Calculated measures and KPIs  
- **MS Excel** – Data inspection and preprocessing  

---

## 📂 Data Pipeline

1. Raw e-commerce order data stored in **PostgreSQL database**
2. Data queried using **SQL**
3. Imported into **Power BI**
4. Cleaned and transformed using **Power Query**
5. Data modeled into relationships
6. Business metrics created using **DAX**
7. Interactive dashboard built for analysis

---
